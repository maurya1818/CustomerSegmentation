# CustomerSegmentation
Customer Segmentation Model | Python, Scikit-Learn
-Developed an unsupervised ML model using K-Means Clustering to categorize customers into 5 distinct behavioral segments based on income and spending scores.

-Applied the Elbow Method and Silhouette Analysis to determine the optimal number of clusters, ensuring mathematical accuracy and model stability.

-Conducted end-to-end data processing, including feature scaling with StandardScaler and data visualization using Seaborn and Matplotlib.

-Derived actionable business insights by profiling segments (e.g., "VIP" vs. "Frugal"), enabling targeted marketing and personalized customer engagement strategies.
